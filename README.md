# AI-Driven Customer Analytics

## Overview

AI-Driven Customer Analytics is a Streamlit-based web application that helps businesses analyze customer behavior using data analytics and machine learning concepts. The system provides customer segmentation, purchase prediction, churn prediction, product recommendations, and interactive visualizations.

## Features

### Dashboard Analytics

* Customer analytics overview
* Interactive business insights
* Visual representation of customer data

### Purchase Prediction

* Predicts customer purchase probability
* Displays prediction results using tables and charts
* Pie chart visualization for purchase analysis

### Customer Segmentation

* Segments customers into different groups using K-Means clustering
* Displays segmented datasets
* Bar charts, pie charts, and scatter plots
* Customer behavior analysis

### Product Recommendations

* Generates personalized product recommendations
* Categorizes customers based on score
* Visual recommendation analysis using charts

### Churn Prediction

* Predicts customer churn risk
* Customer-wise churn analysis
* Dynamic pie chart visualization
* Dataset-based prediction results

## Installation

Clone the repository:

git clone https://github.com/Sushma194/AI-DRIVEN-CUSTOMER-ANALYSIS.git

Move to project folder:

cd AI-DRIVEN-CUSTOMER-ANALYSIS

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
